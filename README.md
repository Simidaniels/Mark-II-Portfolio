# Mark II Portfolio

Mark II Portfolio is a personal showcase project built to present a stronger, more polished front-end design direction through a portfolio homepage and three themed concept projects:

- `CryptoNova` for crypto and fintech design
- `NovaSport` for sports and training branding
- `NovaBike` for futuristic product and e-commerce presentation

This repository is a visual portfolio project built with plain `HTML`, `CSS`, and `JavaScript`, with animated sections powered by `AOS` and icon support from `Boxicons`.

## Overview

This project started as a simpler portfolio concept and was later reworked into a more refined Mark II version. The updated version focuses on:

- cleaner branding
- better layout structure
- stronger typography
- more intentional spacing
- improved mobile responsiveness
- better section flow and user experience

The goal of the redesign was not just to "decorate" the pages, but to re-render them into cleaner, more modern product-style experiences.

## What Was Re-rendered

The project was heavily reworked from the earlier version.

### Before

The earlier build had:

- more template-like layouts
- repeated section patterns
- placeholder copy in several places
- inconsistent visual direction across project pages
- limited or fragile responsiveness
- older navigation behavior and layout issues on smaller screens

### After

The current version was re-rendered with:

- a redesigned portfolio homepage
- a full visual rebrand for `CryptoNova`
- a full visual rebrand for `NovaSport`
- a full visual rebrand for `NovaBike`
- rebuilt mobile navigation on each project page
- improved sticky header and scroll interactions
- clearer content hierarchy
- updated CTA sections, card layouts, and footer structure

In short, the project moved from a rough portfolio collection into a more cohesive front-end showcase.

## Running The Project

This is a static project, so you can run it in any simple way:

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Navigate into the featured project pages from the homepage.

If you prefer, you can also use a local live server in VS Code or any static server tool.

## Design Direction

Each sub-project now has its own visual identity while still feeling connected as part of one portfolio:

- `CryptoNova` uses a cleaner dark fintech aesthetic with cool neon accents.
- `NovaSport` uses a stronger high-energy sports direction with bold contrast.
- `NovaBike` uses a premium product and futuristic mobility style.

The main portfolio was also updated to better present these projects as a curated set rather than unrelated demos.

## Responsiveness

Responsiveness was expanded and rewritten across the portfolio and project pages.

This includes:

- mobile navigation support
- stacked layouts for smaller screens
- responsive grids
- smaller typography scaling
- better spacing on tablets and phones
- contact and CTA sections adjusted for narrow widths

The intent was to make the pages feel designed on mobile, not just "shrunk down."
